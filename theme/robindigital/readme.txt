== Description ==
Robin Digital is a custom WordPress theme built for digital service agencies, consultants, and organizations. The theme features a modern design with Tailwind CSS, customizable sections, and multiple page templates.

== Installation ==
1. Upload the theme files to the `/wp-content/themes/robindigital` directory, or install the theme through the WordPress themes screen directly.
2. Activate the theme through the 'Themes' screen in WordPress.
3. Use the Theme Options page to configure the theme.

== Frequently Asked Questions ==
= How do I set up the home page? =
The home page is automatically set up to use the front-page.php template. You can customize various elements from the Customizer.

= How do I add testimonials? =
Use the Testimonials custom post type to add client testimonials. These will automatically appear in the testimonials section.

= How do I add services? =
Use the Services custom post type to add your service offerings. These will automatically appear in the services section.
